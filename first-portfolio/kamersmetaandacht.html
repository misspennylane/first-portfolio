Frontend Development Project

Als junior frontend developer bij Bee werk ik aan het project Kamers met Aandacht. Dit platform biedt een innovatieve oplossing voor jongeren die op zoek zijn naar een veilige woonplek en een betrokken omgeving. Mijn rol in dit project draait om het creëren van een intuïtieve en aantrekkelijke gebruikerservaring.

Technologie

  • Angular v18
  • Responsive Web Design
  • Frontend Development
  • User Experience (UX) Design

Project Overzicht

Met Angular v18 bouw ik dynamische interfaces die gebruikers gemakkelijk toegang geven tot belangrijke informatie en functies, zoals het zoeken naar beschikbare kamers of het indienen van een aanvraag. Ik werk nauw samen met het backend-team om ervoor te zorgen dat de frontend naadloos aansluit op de onderliggende systemen.

Verantwoordelijkheden

  • Ontwerpen en stylen van responsieve pagina's
  • Implementeren van interactieve componenten
  • Optimaliseren van de gebruiksvriendelijkheid van de webapplicatie
  • Testen en debuggen van code om een foutloze werking te garanderen
  • Werken in een scrum team

Dit project geeft me de kans om mijn vaardigheden in Angular te versterken en een maatschappelijke impact te maken.
