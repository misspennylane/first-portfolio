Frontend Development Project

Als project voor mezelf ontwikkelde ik een cv-website om mijn werkervaring, opleidingen en vaardigheden professioneel en overzichtelijk te presenteren. Hierbij verfijnde ik mijn frontend skills in HTML, CSS en JavaScript.

Technologie

  • HTML5 voor een gestructureerde en semantische opbouw
  • CSS voor een stijlvolle en responsieve vormgeving.
  • JavaScript voor interactieve functies, zoals navigatie en een "back-to-top"-knop.

Project Overzicht

De cv-website biedt een duidelijke en toegankelijke indeling met secties voor werkervaring, opleidingen, vaardigheden en contactinformatie. Het doel was een visueel aantrekkelijke en gebruiksvriendelijke website die als digitaal visitekaartje dient. Het is de bedoeling de website nog verder uit te breiden.
